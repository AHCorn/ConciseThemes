简和从两方面贯彻极简风格，一是代码，二是界面；不需要繁杂的设置，更不需要为各种Bug发愁。

## 使用须知 ##
本主题侧边栏的头像和个性签名需要自己设置。<br>
本主题的评论功能暂时来自其它主题。<br>
极简版不支持评论功能，但将在后续添加。<br>
国内小水管推荐使用普通版，海外小水管推荐极简版。
